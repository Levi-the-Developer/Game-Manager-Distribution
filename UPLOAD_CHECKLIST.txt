     update_required=true, latest_version=1.0.0-alpha.6.
  4. Confirm latest.json download_url points at the release installer above.
  5. Download the release installer and confirm its SHA-256 against the release
     SHA256SUMS.txt asset.

Packaging notes:
  - The alpha installer is unsigned. Windows SmartScreen may warn until a code
    signing certificate is added.
  - update_required is true for alpha.6 so older alpha clients can reach the
    in-app SHA-verified Update Required flow. Alpha.6 itself is not locked because it
    equals latest_version.
  - Master.jsonl carries GAME list_version "1"; future masterlist changes must
    increment this value for the Ready Update action to appear.
  - Keep app source outside this distribution repo unless Levi explicitly
    decides to publish source later.
