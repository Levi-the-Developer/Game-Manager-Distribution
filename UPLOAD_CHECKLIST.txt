Packaging notes:
  - The alpha installer is unsigned. Windows SmartScreen may warn until a code
    signing certificate is added.
  - update_required is true for alpha.7 so older alpha clients can reach the
    in-app SHA-verified Update Required flow. Alpha.7 itself is not locked because it
    equals latest_version.
  - Master.jsonl carries GAME list_version "1"; its applications[] entries add
    OBMM Setup and 4GB RAM Patch Ready buttons without requiring a reinstall.
    Future mod-file/order changes must increment this value for the Ready Update action to appear.
  - Keep app source outside this distribution repo unless Levi explicitly
    decides to publish source later.
