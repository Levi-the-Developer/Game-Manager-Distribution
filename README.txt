Game Manager - GitHub Upload-Ready Payload
==========================================

Public distribution repo for the Game Manager ALPHA channel.

GitHub Pages hosts the game list and update policy the app reads at runtime.
GitHub Releases hosts the public alpha installer.

  channels/alpha/Master.jsonl   The alpha game list (parent/child, schema 2.0).
  updates/alpha/latest.json     Alpha update policy with installer URL.
  checksums/SHA256SUMS.txt      SHA-256 of the Pages-hosted files above.
  UPLOAD_CHECKLIST.txt          Release URLs, checks, and operational notes.

Current public alpha release:
  https://github.com/Levi-the-Developer/Game-Manager-Distribution/releases/tag/v1.0.0-alpha.7

Direct installer URL:
