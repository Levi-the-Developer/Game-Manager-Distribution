Game Manager - GitHub Upload-Ready Payload
==========================================

Staging copy of the files to publish to the public distribution repo for the
ALPHA channel: the game list the app loads and a safe, non-locking update
policy. This is NOT the installer and NOT a release. No app binaries are here.

  channels/alpha/Master.jsonl   The alpha game list (parent/child, schema 2.0).
  updates/alpha/latest.json     Non-locking alpha update policy (no installer URL yet).
  checksums/SHA256SUMS.txt      SHA-256 of the two files above.
  UPLOAD_CHECKLIST.txt          Exact files, destinations, URLs, and checks.

See UPLOAD_CHECKLIST.txt for the step-by-step upload and verification.
